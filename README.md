✈️ SkyBooker - Premium Flight Search Application
SkyBooker is a full-stack flight search platform designed with a focus on high-end user experience, smooth animations, and real-time data integration. This project showcases a modern approach to travel booking interfaces.

🚀 Live Demo
Frontend: skybooker-pro.vercel.app

Backend API: Hosted on Render

🏗️ Technical Architecture
The application is built using a Client-Server architecture, ensuring a clean separation between the user interface and the business logic.

🎨 Frontend (The Experience)
React.js: Used for building a component-based UI and managing application state with Hooks (useState, useEffect).

Tailwind CSS: Employed for rapid, responsive styling and modern UI elements like Glassmorphism.

Custom CSS Animations: Developed specialized keyframes for:

Dynamic Backgrounds: Moving gradients on the login page.

Parallax Flight Effect: A plane icon that flies across the background during search.

Interactive Modals: A specialized "Sky-Pass" boarding pass generator.

⚙️ Backend (The Engine)
Node.js & Express: Powers the API that handles flight search queries.

RESTful API: Developed endpoints to filter and return flight data based on user origin and destination.

Deployment: Hosted on Render to ensure high availability for frontend requests.

🛠️ Key Features
Secure-Feel Login: A beautifully animated login portal featuring floating card effects.

Smart Search: Real-time filtering of flights with instant feedback.

Personalized Boarding Pass: Generates a unique digital ticket with:

Dynamic Passenger Name.

Auto-generated PNR numbers.

Formatted airport codes (e.g., DEL, MUM).

Responsive Design: Fully optimized for both desktop and mobile viewing.

💻 Installation & Setup
If you want to run this project locally, follow these steps:

Clone the repository:

Bash
git clone https://github.com/akrititrivedi18/skybooker-project.git
Install Frontend Dependencies:

Bash
cd skybooker-project/frontend
npm install
Install Backend Dependencies:

Bash
cd ../backend
npm install
Run the Application:

Start Backend: node server.js

Start Frontend: npm start
